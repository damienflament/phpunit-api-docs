in TestSuite at line 125
__construct(mixed $theClass = '', string $name = '')

Constructs a new TestSuite:

  • PHPUnitFrameworkTestSuite() constructs an empty TestSuite.

    • PHPUnitFrameworkTestSuite(ReflectionClass) constructs a TestSuite from the given class.

    • PHPUnitFrameworkTestSuite(ReflectionClass, String) constructs a TestSuite from the given class with the given name.

    • PHPUnitFrameworkTestSuite(String) either constructs a TestSuite from the given class (if the passed string is the name of an existing class) or constructs an empty TestSuite with the given name.
